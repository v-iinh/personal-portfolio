Projects

Portfolio

Pirate Shore

Inspired by The Pirate Bay and other open source projects making film accessible, I developed a free streaming service as well. The site utilizes movie database APIs to get data, and React.js for a seamless user experience.

Sound Mania

Over December break of 2023, I made a Rhythm Game from scratch using Vanilla JS and HTML Canvas. The game comes equipped with 3 gamemodes and levels of difficulty for players to choose from, as well as an effective scoring system.

Self Driving Car

This is my first project involving AI. While following FCC's guide on making an AI car, I improved upon it by making the program interactive and customizable by the user, whereas it can only be edited in the source code originally.

Historical Timelapse

A timeline of maps for major world empires throughout the 15th to 21st century. The website contains GDP and population data for every decade, and has built in zoom features for interactivity. Made in collaboration with Luc Vulliez.

Chef Up

Recipes have never been easier to find. Chef Up consists of search features for a variety of foods, and users are able to save these recipes to revisit at a more convenient time. The responsive design makes it accessible on many devices.

Coin Dash

Blockchain technology is a growing industry as the world continues to dive into a digital revolution. Using the CoinGecko API, users are able to view data for Crypto and NFTs, or search for them as well all within Coin Dash.

The Daily Quill

This page is powered by the News API, providing daily updates, complemented by a functional search system. Built with React, its components and layouts prioritize intuitive design, integrating proper UI & UX practices.

Claustrophobia

Developed with the three.js framework, Claustrophobia is an interactive 3D game that produces an eerie atmosphere. Players are immersed with horror elements of the game, which simulate dark and confined spaces.

STEM Garden Defense

For the Wonderland Hackathon in 2024, I designed and developed a prototype TD game. This was made in a team of 4, and was showcased in the event. The custom timeline and storybuilding also demonstrates our creativity.

Editify

Editify was made with the intention of becoming a universal file editor. The web application allows users to edit and save their files online. As of the moment, the site is made in Vanilla JS, with future improvements to come.

ClassChats+

CC+ is a social network made with Firebase & Vanilla JS, designed with hopes to establish a strong student community in the Boston area. The app also includes a functional moderation system and navigable UI.

Chrominance

Chrominance is a Chrome based browser extension, developed with Vanilla JS and The Color API. This tool makes it convenient for designers to identify colors, and find matching palettes for their projects.

BLS Hack Club

Further demonstrating my working proficiency in web works, BLS Hackclub's site was built by me using JS, CSS, and HTML. We make this in the hopes of inspiring others to learn the courses and use the resources offered by Hackclub.
